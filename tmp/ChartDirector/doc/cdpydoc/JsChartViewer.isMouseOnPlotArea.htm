JsChartViewer.isMouseOnPlotArea

Usage

isMouseOnPlotArea()

Description

Checks if the mouse cursor is on the extended plot area.

See JsChartViewer.setPlotAreaMouseMargin on what is the extended plot area.

Arguments

None

Return Value

True if the mouse is on the extended plot area, otherwise false.
