WebChartViewer.getZoomInWidthLimit

Usage

getZoomInWidthLimit()

Description

Gets the viewport width at maximum zoom in for mouse zoom in actions.

Arguments

None

Return Value

The viewport width at maximum zoom in. See WebChartViewer.setZoomInWidthLimit for the meaning of this value.
