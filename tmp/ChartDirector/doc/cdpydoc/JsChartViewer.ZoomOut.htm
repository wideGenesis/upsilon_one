ChartDirector 6.0 (Python Edition)

JsChartViewer.ZoomOut

Description

A constant equals to 4 to represent using the mouse for zoom out actions.

This constant is used in JsChartViewer.setMouseUsage.

Note: This is a static property (class property). It should be accessed as JsChartViewer.ZoomOut, not objName.ZoomOut.
