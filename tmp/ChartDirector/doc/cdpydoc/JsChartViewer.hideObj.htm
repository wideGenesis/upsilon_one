ChartDirector 6.0 (Python Edition)

JsChartViewer.hideObj

Usage

hideObj(id)

Description

Hides one or more objects drawn by JsChartViewer.

Arguments

ArgumentDefaultDescription
id(Mandatory)The HTML id of the element to be hidden. The special value "crosshair" means that the crosshair cursor will be hidden. The special value "all" means that all lines, text boxes and crosshair cursor drawn by the JsChartViewer object will be hidden.

Return Value

None
