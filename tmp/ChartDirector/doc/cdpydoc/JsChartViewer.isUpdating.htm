JsChartViewer.isUpdating

Usage

isUpdating()

Description

Determines if there is an outstanding partial update (JsChartViewer.partialUpdate or JsChartViewer.partialUpdateAsAttachment) or stream update (JsChartViewer.streamUpdate).

Arguments

None

Return Value

True if there is an outstanding partial update or stream update, otherwise false.
