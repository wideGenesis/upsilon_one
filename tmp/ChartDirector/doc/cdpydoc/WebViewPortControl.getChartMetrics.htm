ChartDirector 6.0 (Python Edition)

WebViewPortControl.getChartMetrics

Usage

getChartMetrics()

Description

Gets the chart metrics of the chart.

The method returns the same value set by a previous call to WebViewPortControl.setChartMetrics.

Arguments

None

Return Value

A text string representing the chart metrics of the chart.
