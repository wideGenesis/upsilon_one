JsViewPortControl.getViewer

Usage

getViewer()

Description

Gets the JsChartViewer object provided by the previous JsViewPortControl.setViewer method call.

Arguments

None

Return Value

The JsChartViewer object provided by the previous JsViewPortControl.setViewer method call.
